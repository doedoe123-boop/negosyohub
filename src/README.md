# NegosyoHub — Backend API

This folder contains the **Laravel 12 backend** powering the NegosyoHub multi-sector marketplace platform.

## Overview

- Provides REST API (`/api/v1/`) for the Vue 3 storefront.
- Hosts Filament admin panels for Admin, Store, and Realty users.
- Handles business logic: orders, commissions, stores, payouts, realty features.
